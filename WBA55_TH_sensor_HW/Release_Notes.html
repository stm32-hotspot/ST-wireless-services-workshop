Release Notes for

WBA55_TH_sensor Hardware Design (KiCad)

Copyright © 2026 STMicroelectronics

Purpose

The WBA55_TH_sensor hardware design provides a reference sensor board based on the STM32WBA55 MCU and the SHT40AD1B temperature and humidity sensor. It is intended to be used together with the BLE_HealthThermometer Application with SHT40AD1B sensor firmware running on STM32WBA55.

This design has been created using the KiCad EDA toolchain and is provided as an example of hardware integration of STM32WBA55 with a digital temperature and humidity sensor and basic supporting circuitry (power supply, connectors, debugging interface, etc.).

The WBA55_TH_sensor hardware design is being delivered as-is, not necessarily supported by ST.

Update History

Main Changes

Initial release of the WBA55_TH_sensor hardware design for STM32WBA55 with SHT40AD1B sensor.

Hardware Overview

  • MCU

    STM32WBA55 series ultra-low-power wireless microcontroller.

  • Sensor

    SHT40AD1B temperature and humidity sensor (I2C interface).

  • Power

    Board powered from CR2032 battery or external power source (as defined in schematic).

  • Connectivity

    Connectivity is defined by STM32WBA55 capabilities - Bluetooth® Low Energy 802.15.4. Design was tuned and tested for Bluetooth® Low Energy and transmitt power 0 dBm.

Components and Documentation

The hardware folder contains:

  • KiCad PCB layout file(s) (*.kicad_pcb).
